# PEDIDO

# FRONT

Install
```
cd front 
npm install
```

Run in development mode
```
npm run dev
```

# API
Install
```
cd api 
npm install
```

Run in development mode

```
npm run dev
```
Run in debug mode
```
npm run debug
```
Migrations
```
migrations:generate
migrations:run
migrations:rever
```
